# 🎵 HZ Music Studio

A full music instrument platform built with HTML, CSS & JavaScript.

## 🎹 Instruments
- **Harmonium** — Sa Re Ga Ma Pa Dha Ni
- **Piano** — Full 3-octave keyboard
- **Guitar** — Real string fretboard
- **Tabla** — Dayan & Bayan with bols

## 🤖 AI Music Assistant
Ask it to play any song on guitar!

## ▶️ How to use
1. Download `HZMusic.exe` → double click to run
2. Or open `HZMusic.html` in any browser

## 🛠️ Built with
HTML · CSS · JavaScript · Web Audio API · Go
